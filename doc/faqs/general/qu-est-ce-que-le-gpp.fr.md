# Qu'est-ce que le Gestionnaire des Processus Projets ?

Le **Gestionnaire des Processus Projets** (GPP) est une application web interne du GIP CAD, destinée à l'instruction des dossiers de candidature déposés par les porteurs de projets de recherche.

## Principe de fonctionnement

Le GPP fonctionne en lien avec la plateforme publique nationale [demarche.numerique.gouv.fr](https://demarche.numerique.gouv.fr/). Le processus se déroule en plusieurs étapes :

1. **Dépôt** : les porteurs de projets déposent leurs dossiers de candidature sur la plateforme publique.
2. **Synchronisation** : les instructeurs du GIP CAD récupèrent ces dossiers dans le GPP via un bouton de synchronisation.
3. **Instruction** : chaque dossier est examiné à l'aide d'une checklist de vérification, avec la possibilité d'ajouter des notes et des pièces jointes pour chaque critère.
4. **Décision** : une fois la vérification terminée, le dossier est accepté, refusé ou classé sans suite.

## Rôles

- **Administrateur** : configure les démarches d'instruction (critères de vérification, liaison avec la plateforme publique, calendrier CSE, synchronisation automatisée).
- **Instructeur** : synchronise les dossiers, effectue les vérifications et rend les décisions.

## Calendrier CSE

Les dossiers sont examinés lors de Comités Sociaux et Économiques (CSE) mensuels. Un dossier doit être déposé au moins 2 semaines avant la date du CSE pour être étudié lors de cette session. Le GPP intègre un calendrier CSE qui permet de visualiser les échéances directement dans le tableau de suivi des dossiers.
