# Comment comprendre mon tableau de bord administrateur ?

Le tableau de bord administrateur regroupe toutes vos démarches d'instruction et permet de les gérer.

## Les sections

- **publiée** : démarches publiées, prêtes à recevoir des dossiers synchronisés.
- **en test** : démarches en cours de configuration, pas encore publiées. Vous pouvez les modifier librement.
- **terminée** : démarches fermées, les dossiers existants peuvent encore être instruits mais aucune nouvelle synchronisation ne sera possible.
- **supprimée** : démarches supprimées.

## Page de gestion d'une démarche

En cliquant sur une démarche, vous accédez à sa page de gestion qui présente :

- **Liaison avec la démarche publique** : le bloc de liaison avec la démarche correspondante sur demarche.numerique.gouv.fr.
- **Indispensable avant publication** : les éléments à configurer (titre, champs du formulaire, instructeurs, etc.).
- **Autres paramètres** : le calendrier CSE, les modèles d'attestation, les emails de notification, etc.

## Actions disponibles

Selon l'état de la démarche, vous pouvez :
- **Publier** une démarche en test.
- **Cloner** une démarche existante pour en créer une variante.
- **Clore** une démarche active.
- **Modifier** les champs du formulaire (les modifications sont appliquées via le système de révisions).
