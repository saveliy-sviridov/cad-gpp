# Comment créer une démarche d'instruction sur le GPP ?

Sur le GPP, une démarche sert de support à l'instruction des dossiers synchronisés depuis la plateforme publique. Elle définit les **critères de vérification** que les instructeurs devront valider pour chaque dossier.

## Étapes de création

1. Connectez-vous en tant qu'administrateur.
2. Cliquez sur **« Créer une nouvelle démarche »** depuis votre tableau de bord.
3. Renseignez les informations générales (titre, description).
4. Dans la section **« Champs du formulaire »**, ajoutez vos critères de vérification.

## Configurer les critères de vérification

Chaque critère de vérification est un champ de type **« Oui/Non »**. Par exemple :

- *Identité du demandeur vérifiée* — L'identité du créateur du dossier a été vérifiée par l'instructeur
- *Données demandées existent dans l'EDS* — L'EDS contient des données permettant de répondre à la demande
- *Contribution du projet au bien commun et pertinence des usages prévus*
- *Faisabilité du projet à partir des données disponibles*
- *Conformité au RGPD et au cadre réglementaire*

Pour chaque champ :
- Choisissez le type **« Oui/Non »**.
- Renseignez un **libellé** clair et une **description** précisant le critère.
- Cochez **« Obligatoire »** si le critère doit être validé pour pouvoir accepter le dossier.

## Publication

Une fois les critères configurés, publiez la démarche. Vous pourrez ensuite la lier à une démarche sur la plateforme publique et commencer à synchroniser les dossiers.
